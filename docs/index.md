# SlideFlow

SlideFlow automates data-driven Google Slides generation from YAML config.

## What you get

- Config-first presentation generation (`slideflow build`)
- Fast validation before execution (`slideflow validate`)
- Data connectors for CSV, JSON, Databricks, and dbt-on-Databricks
- Replacements for text, tables, and AI summaries
- Chart rendering via Plotly, templates, and custom Python functions

## Fast path

1. Set up credentials and install dependencies in [Getting Started](getting-started.md).
2. Run the full local workflow in [Quickstart](quickstart.md).
3. Use [Replacements](replacements.md), [Charts](charts.md), and [Templating](templating.md) to scale.

## Stability policy

SlideFlow maintains backward compatibility for stable configuration and CLI behavior whenever possible.
When behavior changes are required, they are documented in release notes and migration guidance.
